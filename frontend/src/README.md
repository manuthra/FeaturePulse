# FeaturePulse India

FeaturePulse India is an AI-powered product intelligence platform that transforms Google Play Store reviews into actionable product insights. The platform helps product teams identify critical bugs, understand user sentiment, prioritize feature requests, and generate analytics-ready reports in real time.

---

# Key Features

## Review Intelligence
Analyze live Google Play Store reviews using app package IDs.

## AI-Powered Insights
Automatically generate:
- Product summaries
- Critical bug reports
- Feature requests
- User appreciation highlights

## Sentiment Analytics
Interactive visualization for:
- Positive sentiment
- Neutral sentiment
- Negative sentiment

## JIRA-style Ticket Generation
Auto-create engineering-style issue tickets from detected bugs.

## Power BI Export
Export structured CSV datasets for:
- Power BI
- Excel
- Tableau
- Analytics workflows

## PDF Report Generation
Generate downloadable product reports.

## Modern UI/UX
- Dark dashboard interface
- Responsive layout
- Interactive charts
- Professional analytics workflow

# Local Setup

## Clone Repository

```bash
git clone https://github.com/manuthra/featurepulse-india.git
cd featurepulse-india

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

Backend runs on:

```bash
http://localhost:5000
```
